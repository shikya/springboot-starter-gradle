# Spring Hackathon starter pack 

Purpose of the repository:octocat: is to have a base application setup for spring enviornemnt.

| Feature       | Component     |
| ------------- | ------------- |
| Backend Engine | JAVA Spring Boot |
| DataBase | MySql  |
| DB Migration | FlyDB | 
| User Authentication| JWT with custom schema |
